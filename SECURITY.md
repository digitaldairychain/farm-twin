# Security Policy

## Supported Versions

The versions of the project that are currently being supported with security updates are as follows:

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| 0.1.x   | :x:                |

## Reporting a Vulnerability

Please use our [Security Advisories](https://github.com/digitaldairychain/farm-twin/security/advisories) page to privately discuss, fix, and publish information about security vulnerabilities impacting this project.

You may alternatively email [security@farm-twin.org](mailto:security@farm-twin.org) to discuss any issues.
